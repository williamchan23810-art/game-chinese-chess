# 🎮 game-chinese-chess

> A digital two-player mobile and desktop game application based on traditional Chinese chess.

## ✨ Core Features & Gameplay Logic

* **Traditional Rules:** Full traditional Chinese chess logic and piece interactions.
* **Two-Player Support:** Designed seamlessly as a digital two-player mobile game.
* **Safe-Tap Mechanism:** Features a dedicated **three-tap confirmation system**. This validation layout prevents accidental touch errors, ensuring moves are deliberate before they are committed on the board.
* **Responsive Layout:** Optimized interface for smooth play on both mobile screens and desktop monitors.

---

## 🛠️ Built With

* HTML5
* CSS3 (Responsive Grid/Flexbox Layouts)
* Vanilla JavaScript (Game Logic & Validation Rules)

## 📝 How to Play
1. Open the live deployment link above.
2. Select your piece on your turn.
3. Utilize the three-tap confirmation workflow to finalize your strategic moves safely without accidental misclicks.
